# Nexter-Project
Udemy Advanced CSS and Sass: Flexbox, Grid, Animations and More! 강의 공부내용 2탄

### 기술 스택
- HTML5
- CSS
- SCSS
- CSS BEM 방식
- 반응형 디자인
- FlexLayout을 주로 사용

### 실행방법
1. visual studio code 실행
2. index.html에서 마우스 오른쪽 클릭 후 `Open with Live Server` 선택
3. scss 수정 시에는 `npm run compile:sass` 실행 후 저장할 것
